# 📘 Atividade 01 LOPAL

Este repositório contém a resolução da **Atividade 01** da disciplina de **Lógica de Programação e Algoritmos (LOPAL)**, focada no desenvolvimento de lógica através de fluxogramas.

---

## 🎯 Objetivos
* Resolução de problemas com raciocínio lógico.
* Construção de passo a passos para realizar tarefas.

## 🛠️ Ferramentas Utilizadas
* **Flowgorithm** (Criação dos fluxogramas)
* **Git / Github** (Controlo de versão e portfólio)
